What tools did I use to make this web page?

  • Just as in every web page, I used HTML for the content creating process

  • I used CSS for styling each element of the page

  • I also used CSS flex for every card image, the navigation bar and the banner

  • In the card container I used CSS grid, adapting the columns for a perfect experience in eevery device

  • I used vanilla javascript, to make a searcher, the interaction with the navigation bar, the goUp button and some other little things

  • As this web page has many images, I had to optimize each image to the maximum, changing its size and converting it to webp format. Also, for a better optimization I used lazy loading with the javascript API Intersection Observer

  • And finally I used Inkscape, an open source vector graphics editor to create the icons of this list an the favicon
