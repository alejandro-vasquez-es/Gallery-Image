What tools did I use to make this web page?

  • As every web page, i used HTML for the content

  • I used CSS for styling each element of the page

  • I also used CSS flex for every card image, the navigation bar and the banner

  • In the card container i used CSS grid, adapting the columns for every device

  • I used vanilla javascript, to make a searcher, the interaction with the navigation bar, the goUp button and other little things

  • As this web page has many images, I had to optimize each image to the maximum, changing sizes and converting it to webP format. And for more optimization i used lazy loading with the javascript API Intersection Observer

  • And finally i used Inkscape, an open source vector graphics editor to make the icons of this list an the favicon
